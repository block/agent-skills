Things you can do in edit mode

  • Click any text and type — it autosaves.
  • 🖼 Add an image (or paste with ⌘/Ctrl+V) and drag it.
  • T Add a free-floating text box.
  • ↶ Undo / ↷ Redo with ⌘/Ctrl+Z.
  • ▦ Open the filmstrip and drag to reorder slides.
  • ⬇ Export a clean standalone HTML.

That's it.

Two files — editor.js + editor.css — layered on top of your deck. No build, no dependencies. Now go ship.
